# 📊 Tableau Full Course Projects (Baraa)

This repository contains my hands-on projects, exercises, and dashboards created while following the **Tableau Full Course by Baraa** on YouTube.

The goal of this repository is to document my learning journey and demonstrate my skills in data visualization, dashboard development, and data storytelling using Tableau.

---

## 🚀 What I Learned

Throughout this course, I gained knowledge and practical experience in:

- Data visualization principles
- Tableau interface and workflow
- Connecting and preparing datasets
- Creating charts and graphs (bar, line, pie, maps, etc.)
- Building interactive dashboards
- Using filters, parameters, and calculated fields
- Data storytelling and presentation
